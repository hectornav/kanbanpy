# 🗂️ kanbanpy

**kanbanpy** is a simple Kanban-style desktop application built with **PyQt6**, designed to help individuals stay focused and organized while managing tasks.

## ✨ Features

- Create, update, and delete tasks
- Drag and drop tasks across **To Do**, **Doing**, and **Done** columns
- Persistent task state between sessions
- Lightweight and user-friendly interface

## 📦 Installation

A precompiled `.deb` package is available for Ubuntu-based systems:

```bash
sudo dpkg -i kanban-app_1.0-1_all.deb
